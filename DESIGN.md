# Design

Визуальная система «Алтын Юрта» — **editorial-luxury food-design**. Тёплый кремовый
фон доминирует, тёмные фото-блоки дают вес и контраст, terracotta/coral — точечный
акцент только на действиях и ключевых метках. Засечковый display-шрифт на заголовках,
чистый гротеск на теле текста, пилюльные (pill) кнопки. Источник истины по значениям —
[`src/styles/tokens.css`](src/styles/tokens.css); полный аудит контраста —
[`artifacts/03_design-system/contrast.md`](artifacts/03_design-system/contrast.md).

> Замысел сознательно НЕ «Золотая юрта» (войлок/пергамент/сургуч) — отказ по явному
> запросу. Тепло несут цвет и съёмка блюд, не этнический декор.

## Theme

Светлая тёплая editorial-тема. Стратегия цвета — **restrained → committed**: тёплый
кремовый фон ~55–60%, тёмные панели ~10–15%, terracotta-акцент ≤10% (60/30/10 в
адаптации). Акцент никогда не заливает крупные секции — только точка действия.

## Color

| Токен | Hex | Роль |
|---|---|---|
| `--color-bg` | `#F4EBDD` | основной фон страниц (~55–60% площади) |
| `--color-surface` | `#FBF5EC` | карточки, секции на bg (~20%) |
| `--color-text` | `#2A1F1A` | основной текст, тёплый почти-чёрный |
| `--color-muted` | `#6B5C4F` | вторичный текст, подписи |
| `--color-accent` | `#C2542F` | terracotta/coral — фон CTA, бейджи, крупные акценты |
| `--color-accent-hover` | `#A8431F` | accent-текст на светлом + hover кнопок |
| `--color-accent-2` | `#1C1410` | тёмные фото-панели, footer, оптовый hero |
| `--color-on-accent` | `#FFFFFF` | текст на `accent` (кнопки) |
| `--color-on-accent-2` | `#F4EBDD` | текст на тёмных панелях |
| `--color-border` | `rgba(42,31,26,0.08)` | тонкие обводки 1px |

**Правила контраста (WCAG AA, измерено):**
- `text` на `bg` — **13.6:1**; `muted` на `bg` — **5.44:1**; `bg` на `accent-2` — **15.4:1**.
- `on-accent` (#FFFFFF) на `accent` — **4.56:1** (проходит; на кнопках держим bold/крупный кегль как запас).
- ⚠️ `accent` как цвет текста на `bg` — **3.86:1**: только для крупного текста/иконок (≥3:1).
  Для мелкого accent-текста (метки, цены, активное меню, подсказка-стимул) — `accent-hover` (**5.10:1**).
- Серый текст на цветном фоне не использовать — брать затемнение того же тона или прозрачность текста.

## Typography

> Cyrillic-first. Fraunces / General Sans отвергнуты (нет кириллицы). Inter в фолбэках запрещён.

- **Display (заголовки):** `'Cormorant', Georgia, 'Times New Roman', serif` —
  высококонтрастный редакционный сериф с полной кириллицей. На экране тонок →
  базовый вес заголовков **600**, letter-spacing **-0.02em**. Цвет заголовков —
  `text` или `accent-2` (НЕ `accent`: тонкие штрихи провалят контраст).
- **Body (текст):** `'Manrope', system-ui, -apple-system, 'Segoe UI', sans-serif` —
  геометрико-гуманистический гротеск с кириллицей. Контраст пары: serif + sans.
- **Шкала:** 1.250 (major third), база 16px.
  - text: `--text-xs` 12 / `--text-sm` 14 / `--text-base` 16 / `--text-lg` 20 / `--text-xl` 25 / `--text-2xl` 31
  - display: `--display-sm` 25 / `-md` 31 / `-lg` 39 / `-xl` 49 / `-2xl` 61 / `-3xl` 76 (верхние ступени — hero)
- Подключение — Google Fonts в [`index.html`](index.html) (Cormorant ital,wght 400–700; Manrope 400–700).
- Длина строки тела — 65–75ch; `text-wrap: balance` на h1–h3.

## Spacing & Layout

- **Базовый шаг 8px.** Все отступы кратны 8: `--space-1..12` = 8/16/24/32/48/64/96.
  ⚠️ Tailwind `--spacing` оставлен дефолтным (4px) — 8px-сетка держится дисциплиной
  чётных шагов, НЕ глобальным override (override ломал высоты кнопок/полей вдвое, см. review).
- **Контейнер:** `--container-max` 1200px, поля 24px (мобильный) / 48px (десктоп).
- **Сетка:** 12 колонок, gap 24px. Flexbox для 1D, Grid для 2D; адаптивные сетки —
  `repeat(auto-fit, minmax(280px, 1fr))`.

## Shape, Radius, Shadow

- **Радиусы:** карточки `--radius-card` 16px, изображения `--radius-image` 20px,
  кнопки `--radius-pill` 999px (pill — ключевая деталь референса).
- **Тени:** мягкие тёплые (не серые) — `--shadow-card` `0 12px 32px rgba(28,20,16,0.12)`
  для карточек над bg/surface. Тёмные фото-блоки — без тени (вес несут цветом).
- **Обводки:** `1px` `--color-border` там, где нужна лёгкость (карточка товара в каталоге).

## Components

- **CTA «Заказать» / «Оформить заказ»:** pill-кнопка, фон `accent`, текст `on-accent`
  (#FFFFFF), hover — `accent-hover`.
- **Карточка товара:** фон `surface`, радиус 16px, фото ~4:3 сверху, название (display,
  малый кегль), цена (`accent-hover`, bold), состав/срок годности (`muted`, мелкий кегль).
- **Блок условий корзины** (мин. сумма / порог бесплатной доставки / оплата при
  получении): тёмная плашка `accent-2`, текст `on-accent-2` — намеренно выделен.
- **Подсказка-стимул** («добавьте ещё N ₽ — бесплатная доставка»): `accent-hover`.
- **Блок доверия / сертификаты:** на `surface`, иконки/значки `accent`, текст `text`.
- **Страница «Опт»:** тёмная hero-плашка `accent-2`, заголовок `on-accent-2` —
  разводит B2B-канал от светлого розничного пути.
- shadcn/ui-примитивы (button/badge/card/input/label/textarea) перекрашены на токены;
  остаточные shadcn-переменные (primary/secondary/destructive/ring/input) не используются.

## Motion

- Reveal-компонент (`src/components/Reveal.tsx`) для stagger-входов. Ease-out
  экспоненциальные кривые, без bounce/elastic.
- Reveal должен усиливать уже видимый дефолт (не гейтить видимость контента классом).
- `prefers-reduced-motion: reduce` — обязательная альтернатива (кроссфейд/мгновенно).

## Stack & Tokens

- **Стек:** Vite + React (TypeScript), shadcn/ui + Tailwind v4. SPA, точка входа
  `index.html` → `src/main.tsx`. Маршруты: `/`, `/catalog`, `/cart`, `/checkout`, `/wholesale`.
- **Токены:** CSS custom properties — [`src/styles/tokens.css`](src/styles/tokens.css)
  (внедрённый источник истины) + W3C JSON — [`artifacts/03_design-system/tokens.json`](artifacts/03_design-system/tokens.json).
- **Инвариант:** хардкод hex/px в компонентах запрещён — только токены.

## Do / Don't

- ✅ `accent` — фон кнопок/бейджей и крупные акценты; `accent-hover` — accent-текст мелкого кегля.
- ✅ Тёмные панели `accent-2` несут вес сами (без тени); кремовый фон доминирует.
- ✅ Условия заказа — в корзине, видимы до оформления.
- ❌ `accent` как цвет мелкого текста (3.86:1) и как заливка крупных секций.
- ❌ Серый текст на цветном фоне; шрифты без кириллицы; элементы прежней «Золотой юрты».
- ❌ Глобальный override `--spacing` (ломает Tailwind-шкалу высот).
