# Алтын Юрта — мобильная версия сайта

Статичная страница-превью (HTML + CSS + JS в одном файле). Зависимостей нет,
кроме шрифтов Google Fonts и фото с Unsplash (грузятся по сети).

## Файлы
- `index.html` — сама страница (точка входа).
- `hero.mp4` — фоновое видео в hero.
- `poster.jpg` — кадр-заглушка видео (показывается, пока ролик грузится).

## Как опубликовать на GitHub Pages
1. Создайте репозиторий на GitHub (например `altyn-yurta`).
2. Загрузите в него все файлы из этого архива (index.html, hero.mp4, poster.jpg)
   в корень репозитория.
3. Settings → Pages → Build and deployment → Source: **Deploy from a branch**.
4. Branch: `main`, папка `/ (root)` → Save.
5. Через 1–2 минуты сайт будет доступен по адресу
   `https://<ваш-логин>.github.io/<имя-репозитория>/`.

> Видео автозапускается без звука (muted) и зациклено — так требуют браузеры
> для автоплея. На мобильных работает в фоне hero.
